## ⑤工夫した点・こだわった点
- じゃんけんと水槽育成、２つのゲームを相関的に作成
- カウントダウンの時間をCSSで調整し、ゆったりした感覚を付与
- BGM、効果音はオン・オフ可能な設計（公開にあたって音源は削除）

## ⑥難しかった点・次回トライしたいこと（又は機能）
- 水槽の外でもアイテムを行使できてしまう不具合を直したが、だいぶ苦戦した（「名前空間」を勉強）
- コメント化しながら全体的に理解して取り組めたが、コードはコメントだらけ
- 時間切れになった後の更なる展開に着手する余裕はなかった
- レスポンシブデザインに着手する余裕はなかった
